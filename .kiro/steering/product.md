# Product Overview

**Saga** is an AI-powered family biography platform for intergenerational storytelling.

## Core Features

- **Story Recording**: Audio recording with AI-powered transcription and content generation
- **Multi-language Support**: 8 languages (en, zh-CN, zh-TW, ja, ko, es, fr, pt)
- **Project Management**: Collaborative family biography projects with role-based access
- **Resource Management**: Seat-based subscription model (facilitator/storyteller seats, project vouchers)
- **AI Integration**: OpenAI for content generation, Google Cloud Speech for transcription
- **Story Discovery**: Search, bookmarks, recommendations, and sharing features

## User Roles

- **Owner**: Project creator with full administrative access
- **Facilitator**: Can manage stories and invite storytellers
- **Storyteller**: Can record and manage their own stories

## Key User Flows

1. Create project → Invite members → Record stories → Generate AI content → Export/share
2. Purchase resources → Allocate seats → Manage subscriptions
3. Browse stories → Bookmark → Share → Discover recommendations
